# 🪙 مشروع الصائغ | Alsayegh Project

تطبيق تقني متكامل مصمم خصيصاً لخدمة قطاع التعدين. يهدف المشروع إلى توفير أدوات حسابية دقيقة لأسعار الذهب والأوزان لحظة بلحظة، مع مراعاة ظروف العمل في المناطق النائية.

## 🚀 الأهداف الحالية:
* حساب سعر الذهب بناءً على البورصة العالمية وسعر الصرف المحلي.
* تطوير واجهة مستخدم تعمل على الهواتف الذكية (Android).
* توفير قاعدة بيانات سحابية لحفظ العمليات الحسابية.

## 🛠️ التقنيات المستخدمة:
* **Python**: اللغة الأساسية للكود.
* **Kivy / Buildozer**: لتحويل المشروع إلى تطبيق أندرويد.
* **GitHub**: لإدارة النسخ وتوثيق العمل.
